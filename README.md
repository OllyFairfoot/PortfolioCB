# Side-scrolling portfolio

## Files to upload to GitHub

Upload all of these files and folders to the root of your `portfolio` repository.

## Add an artwork

1. Put the finished image in `images/artworks/`.
2. Put process photographs in `images/progress/`.
3. Open `artworks.js`.
4. Copy one artwork object and edit its:
   - id
   - title
   - year
   - medium
   - dimensions
   - image filename
   - x and y position
   - displayed width and height
   - description
   - progress images and captions

## Publish with GitHub Pages

In the repository, open:

Settings → Pages → Build and deployment → Deploy from a branch

Choose the `main` branch and `/ (root)`, then save.
